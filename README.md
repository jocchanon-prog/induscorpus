The 417 Indus script symbols used in this corpus are sourced from the IM-417-150 dataset on Kaggle , created by storesource and updated in 2024.

Dataset Details
Attribute	Information
Source	IM-417-150 Kaggle Dataset
Original Reference	Iravatham Mahadevan, "The Indus Scripts: Texts, Concordance and Tables", Archaeological Survey of India (1977), Pages 32-34
Images	417 grayscale images, 150×150 pixels each
Format	PNG files and CSV with flattened pixel representations (22500 pixels per image)
License	Apache 2.0

Attribution
This corpus builds upon the foundational work of Iravatham Mahadevan (1977) and the digitization efforts of the Kaggle community. The original dataset contains 417 characters extracted from images of seals from the Indus Valley Civilization.

## 📥 Getting the Dataset

1. **Download the original images** from [Kaggle](https://www.kaggle.com/datasets/storesource/im-417-150)
2. Extract the PNG files to a folder (e.g., `./indus_images/`)
3. Run the generator and select this folder

The dataset includes:
- 417 PNG images (150×150 grayscale)
- CSV file with flattened pixel representations
- Character labels M001–M417 corresponding to Mahadevan's concordance
